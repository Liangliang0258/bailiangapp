# 舒尔特方格 (Schulte Grid)

> 专业的注意力训练和专注力提升工具

舒尔特方格是一种经典的认知训练方法，通过在网格中按顺序快速点击数字来训练注意力和专注力。该方法被广泛应用于教育学、心理学和运动训练领域。

## 功能特点

- 多种难度级别（3×3、4×4、5×5、6×6、4×5）
- 精确到毫秒的计时功能
- 本地存储最佳成绩和历史记录
- 音效反馈（正确/错误/完成提示）
- 流畅的动画效果
- 响应式设计，支持移动端

## 如何使用

### 在线使用

直接打开 `index.html` 文件即可在浏览器中运行。

### 游戏规则

1. 选择一个难度级别（网格大小）
2. 点击"开始游戏"按钮
3. 按照数字顺序（1, 2, 3...）快速点击网格中的数字
4. 完成所有数字点击后，系统会显示您的用时
5. 尝试打破自己的最佳记录！

## 技术栈

- 纯 HTML5 / CSS3 / JavaScript（无框架依赖）
- Web Audio API 生成音效
- LocalStorage 数据持久化
- CSS Grid 响应式布局

## 浏览器兼容性

支持所有现代浏览器：
- Chrome / Edge 90+
- Firefox 88+
- Safari 14+

## 训练建议

- **初学者**：从 3×3 或 4×4 网格开始
- **进阶**：尝试 5×5 或 6×6 网格
- **每日训练**：建议每天练习 10-15 分钟
- **目标参考**：4×4 网格优秀水平约为 30 秒以内

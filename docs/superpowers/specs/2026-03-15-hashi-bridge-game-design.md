## 1. 游戏概述

### 1.1 游戏概念
数桥（Hashiwokakero）是一款经典逻辑推理益智游戏。玩家需要在网格上绘制桥梁连接岛屿，使每个岛屿的桥梁数量等于其中心显示的数字。

### 1.2 游戏目标
通过逻辑推理，在所有岛屿之间建立桥梁网络，满足：
- 每个岛屿的桥梁数量等于其数字
- 所有岛屿通过桥梁网络连通
- 桥梁只能水平或垂直连接

### 1.3 目标用户
- 成年人
- 喜欢逻辑推理游戏
- 短时间休闲（1-3分钟/局）

## 2. 核心游戏机制

### 2.1 数据结构

```javascript
gameState = {
  // 网格配置
  grid: {
    width: 5,      // 简单: 5x5, 中等: 7x7, 困难: 9x9
    height: 5,
    islands: [
      { row: 0, col: 1, bridges: 0, target: 2 },
      { row: 2, col: 3, bridges: 0, target: 3 },
      // ...
    ]
  },

  // 已建的桥梁
  bridges: [
    { from: {row: 0, col: 1}, to: {row: 0, col: 3}, count: 1 },
    // ...
  ],

  // 游戏状态
  difficulty: 'medium',  // easy | medium | hard
  moves: 0,
  startTime: null,
  endTime: null,
  isComplete: false
}
```

### 2.2 交互规则

**点击序列：**
1. 用户点击岛屿A → 高亮可连接的邻居（虚线）
2. 用户点击邻居B → 在A和B之间建1桥
3. 用户再次点击B → 建第2桥
4. 用户再次点击B → 删除桥（回到0桥）

**桥梁规则：**
- 桥梁只能水平或垂直连接相邻岛屿
- 两岛之间最多2座桥
- 桥梁不能交叉

**胜利条件：**
1. 所有岛屿的当前桥数 = 目标数字
2. 所有岛屿通过桥梁网络连通

### 2.3 关卡生成算法

```
1. 根据难度选择网格大小
   - Easy: 5x5, 4-6个岛屿
   - Medium: 7x7, 6-9个岛屿
   - Hard: 9x9, 8-12个岛屿

2. 生成合法解
   - 随机放置岛屿
   - 生成满足规则的桥梁网络
   - 确保所有岛屿连通

3. 反推岛屿数字
   - 统计每个岛屿的桥梁数
   - 设置为目标数字

4. 清空桥梁，交给玩家
```

### 5.4 核心算法

**连通性检测（并查集）：**
```javascript
function checkConnectivity(islands, bridges) {
  const uf = new UnionFind(islands.length);

  bridges.forEach(bridge => {
    const id1 = getIslandId(bridge.from);
    const id2 = getIslandId(bridge.to);
    uf.union(id1, id2);
  });

  return uf.getCount() === 1; // 所有岛屿连通
}
```
